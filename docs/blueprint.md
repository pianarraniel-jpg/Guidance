# **App Name**: GuidanceSync

## Core Features:

- User Authentication & Authorization: Secure login for students, counselors, and administrators with robust role-based access controls to dedicated application sections, utilizing a custom LocalStorage service for mock user management.
- AI-Powered Stress Assessment Chatbot: An interactive chatbot tool to guide students through confidential stress assessment questionnaires and capture their wellness check-in responses.
- Student Appointment Booking & Management: Students can intuitively view counselor availability, then book, reschedule, or cancel 'Kamustahan' (wellness check-in) sessions.
- Counselor Session Management Dashboard: Counselors gain access to student pre-session summaries, manage their weekly schedules, and securely record confidential post-session notes.
- Integrated Messaging System: A direct, secure messaging interface facilitating private communication and follow-ups between students and their assigned counselors.
- Administrator Analytics & Reporting: An administrative dashboard providing a high-level overview of system usage, user engagement metrics, and generated reports for institutional oversight.

## Style Guidelines:

- Primary calls to action and structural elements will utilize a deep, professional Teal (#248F7D), symbolizing support, health, and academic precision.
- The interface will feature a light, serene background (#F0F5F4), promoting focus and a sense of calm during interactions.
- Accents, warnings, and highlighting elements will employ a vibrant Amber (#F4AF25), adding a touch of warmth and drawing attention where needed.
- All text throughout the application, including headlines and body content, will use the 'Inter' sans-serif typeface, selected for its modern, highly legible, and neutral aesthetic.
- Utilize Lucide React for iconography, ensuring a consistent set of clean, simple, and easily understandable icons that reinforce navigation and actions without visual clutter.
- The application features a clean, organized layout with clearly segregated role-based interfaces (Student, Counselor, Admin), employing semantic HTML elements and a consistent 4px/8px base spacing system for visual hierarchy.
- Subtle and purposeful animations, such as smooth transitions for state changes and form submissions, will enhance user feedback and overall interface responsiveness without distracting from core tasks.
